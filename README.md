# mylocalvimconf

This is my local vim config

### Install
First of all it's use **vim-plug** and **vim** not anything else , vim-plug is pakage manager for vim

```
git clone https://github.com/SepehrImanian/mylocalvimconf.git
cd mylocalvimconf
chmod 775 install.sh
./install.sh
```
